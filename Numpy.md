# 📝 NumPy (Numerical Python) 핵심 이론 정리

데이터 분석과 머신러닝의 가장 기본이 되는 파이썬의 핵심 라이브러리, **NumPy(넘파이)**의 핵심 개념을 정리한 문서입니다.

---

## 1. NumPy란?
파이썬에서 **대규모 다차원 배열과 행렬 연산**을 쉽고 빠르게 처리할 수 있도록 돕는 라이브러리입니다. 데이터 사이언스, 머신러닝(딥러닝), 시각화 등 모든 데이터 분석 라이브러리(Pandas, Scikit-learn, TensorFlow 등)의 기반이 됩니다.

### 왜 일반 파이썬 리스트 대신 NumPy를 쓸까?
* **압도적인 속도 (C언어 기반):** 내부적으로 C언어로 구현되어 있어 연산 속도가 파이썬 리스트에 비해 훨씬 빠릅니다.
* **벡터화 연산 (Vectorization):** 반복문(`for`문)을 쓰지 않고도 배열 전체에 대한 연산을 한 번에 처리할 수 있습니다.
* **메모리 효율성:** 파이썬 리스트와 달리 연속된 메모리 공간에 데이터를 저장하여 메모리를 효율적으로 사용합니다.

---

## 2. 핵심 개념: ndarray (N-dimensional Array)
NumPy의 가장 중추적인 객체는 **`ndarray`(다차원 배열)**입니다.

> **💡 단 하나의 철칙:** 
> 파이썬 리스트는 다양한 타입의 데이터를 섞어서 넣을 수 있지만, NumPy의 `ndarray` 내부의 모든 원소는 **반드시 같은 데이터 타입(Single Type)**이어야 합니다.

### 배열의 차원 이해하기
* **1차원 배열 (Vector):** 하나의 행 또는 열로 이루어진 형태 `[1, 2, 3]`
* **2차원 배열 (Matrix):** 행(Row)과 열(Column)로 이루어진 표 형태 (가장 많이 사용됨)
* **3차원 이상 배열 (Tensor):** 행, 열에 깊이(Depth)가 추가된 고차원 형태 (이미지 데이터 처리 등에 필수)

---

## 3. 핵심 기능 및 연산 규칙

### ① 브로드캐스팅 (Broadcasting)
💡 **넘파이의 꽃이라고 불리는 기능입니다.**
형태(Shape)가 다른 배열끼리 연산할 때, NumPy가 내부적으로 배열의 크기를 맞춰서 연산이 가능하도록 확장해 주는 기능입니다.

* 원래 행렬 연산을 하려면 두 행렬의 크기가 같아야 하지만, 브로드캐스팅 덕분에 `(3, 3)` 행렬에 단일 숫자 `5`를 더하면 모든 원소에 `5`가 더해집니다.

### ② 벡터화 연산 (Element-wise Operation)
배열의 각 요소별로 연산이 링크되어 한 번에 계산됩니다.
```python
import numpy as np

arr = np.array([1, 2, 3])
print(arr * 2)  # 결과: [2, 4, 6] -> for문 없이 한 번에 계산!
